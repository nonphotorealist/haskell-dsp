Description

Simple module for generating Maclaurin series representation of a few functions:

f(x) = sum [ a_i * x^i | i <- [0..] ]

The Int parameter for all functions is the order of the polynomial, eg:

[ a_i | i <- [0..N] ]

and not the number of non-zero terms
